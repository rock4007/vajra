üöÄ Quick Start

Method 1: Direct Browser (Easiest)

1

Double-click app.html

The app opens immediately in your default browser

2

Allow location permissions

Click "Allow" when prompted for location access

3

You're ready!

All features available including offline mode

Method 2: Web Server (Recommended)

1

Windows: Run start.bat

Double-click start.bat or run:

start.bat
2

Linux/macOS: Run start.sh

Make executable and run:

chmod +x start.sh && ./start.sh
3

Open browser

Navigate to: http://localhost:8000/app.html

Method 3: Manual Python Server

cd VajraBackend python -m http.server 8000 # Then open http://localhost:8000/app.html
